# Additional Requirements

1. Each Python file must contain a main function that can be executed individually.
2. Include comments and docstrings in your Python files for better readability.
3. Use proper exception handling in your code.
4. Follow PEP 8 guidelines for Python code style.
5. Ensure all dependencies are listed in the \`requirements.txt\` file.
6. Include a brief description of the purpose of each folder and Python file in the README.txt.
